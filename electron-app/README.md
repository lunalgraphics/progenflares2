# Progen Flares 2 — Electron App

Standalone desktop application wrapping the Progen Flares 2 web editor in an Electron shell. Provides a native window experience without requiring a browser.

## Requirements

- Node.js 18+ for building
- The web bundle must be built first (see below)

## Building

### 1. Build the web bundle

From the repository root:

```bash
npm run build:electron
```

This outputs the Svelte app to `electron-app/app/build/`.

### 2. Package the Electron app

```bash
cd electron-app
npm install
npm run build
```

Platform-specific builds:

| Command | Target |
|---------|--------|
| `npm run build:win32` | Windows (portable .exe) |
| `npm run build:darwin` | macOS (.zip) |
| `npm run build:linux` | Linux (.deb) |
| `npm run build` | All platforms |

Built artifacts appear in `electron-app/dist/`.

## Development

To run the app without packaging:

```bash
cd electron-app
npm start
```

This opens the app from the local `app/` directory. Make sure you've already run `npm run build:electron` from the root to populate the bundle.

## File Structure

```
electron-app/
├── main.js                 Electron main process (window creation)
├── package.json            Electron app manifest + electron-builder config
├── resources/              Build resources (icons, etc.)
└── app/                    Web content loaded by the Electron window
    ├── index.html          Entry HTML
    ├── icon.png            App icon
    └── build/              Built Svelte bundle (generated by build:electron)
        ├── bundle.js
        └── bundle.css
```

## Configuration

The electron-builder config in `package.json` defines:

- **Windows**: Portable executable (no installer required)
- **macOS**: .zip archive
- **Linux**: .deb package

App ID: `com.electron.lunalgraphics-progenflares2`

## Notes

- DevTools are only available in non-packaged (development) builds
- The menu bar is hidden for a cleaner UI
- The window maximizes on launch
- On macOS, the app stays open when all windows are closed (standard macOS behavior)
